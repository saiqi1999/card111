# Cardgame111 - 卡牌游戏项目

这是一个使用Godot 4.4.1引擎开发的卡牌游戏项目。

## 项目结构

- `assets/` - 游戏资源文件夹
  - `images/` - 图像资源
- `scenes/` - 游戏场景文件夹
  - `root.tscn` - 主场景
- `scripts/` - 游戏脚本文件夹
  - `cards/` - 卡牌系统相关脚本
  - `mobs/` - 怪物相关脚本
  - `button_handler.gd` - 按钮事件处理脚本

## 功能概述

### 卡牌系统

项目实现了基础的卡包系统：

- `CardPackBase` - 卡包基类，提供卡牌管理的基本功能
- `StrikeCardPack` - 打击卡包，继承自卡包基类

### 用户界面

- 主场景中包含一个可点击的按钮，点击后会在控制台打印信息

## 如何运行

1. 使用Godot 4.4.1或更高版本打开项目
2. 点击编辑器中的运行按钮或按F5启动游戏
